# PROG3050.Tests
## Setup
```bash
cd PROG3050.Tests
pip install -r requirements.txt
```

## How to run a test
### Option 1
1. Select a test file to run (`test_*.py`)
2. Use VScode Run & Debug

### Option 2
1. Select a test file to run (`test_*.py`)
2. Run in the terminal
```bash
cd PROG3050.Tests
python test_login.py # or python3 test_login.py 
```
